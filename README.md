## Creazione di grafici per interpretare i dati

### 📦 Librerie:
    # matplotlib
    # seaborn (opzionale)
    import matplotlib.pyplot as plt
    
    x = [1, 2, 3]
    y = [10, 20, 30]
    
    plt.plot(x, y)
    plt.title("Grafico vendite")
    plt.xlabel("Tempo")
    plt.ylabel("Valori")
    plt.show()
